# Apex Legal Partners Website

## Project info

A professional website for Apex Legal Partners, a full-service law firm specializing in corporate law, litigation, real estate, intellectual property, and estate planning.

## Development

**Local Development**

To work on this project locally, you can clone this repo and make changes.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd apex-legal-partners

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
